📨 LISTA DE MENSAGENS DO SISTEMA BEACONFLY

Versão: 2.0.0
Data: 2026-04-19
Uso: Referência para implementação da Ground Station

# 1. MENSAGENS DE SISTEMA (MSG_*)

ID (Hex)	Constante	Direção	Descrição	Payload
0x10	MSG_HEARTBEAT	ESP1 → GS	Coração do sistema. Indica que o ESP1 está vivo. Enviado a cada 1 segundo.	Timestamp (opcional)
0x11	MSG_TELEMETRY	ESP2 → GS	Telemetria normal do sistema. Dados filtrados (10Hz). Contém campos TLV com estado do sistema.	Lista TLV com FLD_ROLL, FLD_PITCH, FLD_ALT_FUSED, FLD_BATT_V, etc.
0x12	MSG_COMMAND	GS → ESP1	Comando da Ground Station para o ESP1. Pode ser CMD_ARM, CMD_SET_ROLL, etc.	ID do comando + parâmetros
0x13	MSG_ACK	ESP1 → GS	Confirmação de receção de comando. Opcional (pode ser desativado).	ID do comando confirmado
0x14	MSG_FAILSAFE	ESP1/ESP2 → GS	Notificação de ativação/desativação do modo failsafe. Prioridade SUPER_CRITICAL.	Razão (FLD_FS_REASON) + Ação (FLD_FS_ACTION)
0x15	MSG_DEBUG	ESP1/ESP2 → GS	Mensagens de debug para desenvolvimento. Prioridade dinâmica (SUPER_LOW ou SUPER_CRITICAL).	Texto livre
0x16	MSG_VIDEO	ESP1 → GS	Pacote de vídeo (streaming experimental). Fragmentado em chunks.	FLD_VIDEO_FRAME_ID, FLD_VIDEO_CHUNK_ID, FLD_VIDEO_TOTAL, FLD_VIDEO_PAYLOAD
0x17	MSG_SHELL_CMD	GS ↔ ESP1/ESP2	Comando de shell remoto. Permite diagnóstico e configuração remota.	Comando em texto (ex: "status", "get_roll")
0x18	MSG_SI_DATA	ESP2 → ESP1	Dados em unidades SI (Système International). Enviados do ESP2 para o ESP1 para controlo de voo.	FLD_ROLL, FLD_PITCH, FLD_YAW, FLD_ROLL_RATE, FLD_PITCH_RATE, FLD_YAW_RATE, FLD_ALT_FUSED, FLD_VX, FLD_VY, FLD_VZ, FLD_BATT_V, FLD_BATT_A

# 2. CAMPOS TLV (DENTRO DAS MENSAGENS)

## GPS / Navegação

ID (Hex)	Constante	Tipo	Descrição	Unidade
0x20	FLD_GPS_LAT	int32_t	Latitude	graus × 10⁷
0x21	FLD_GPS_LON	int32_t	Longitude	graus × 10⁷
0x22	FLD_GPS_ALT	int32_t	Altitude GPS	mm
0x23	FLD_GPS_SPEED	uint16_t	Velocidade GPS	cm/s
0x24	FLD_GPS_SATS	uint8_t	Número de satélites	-
0x25	FLD_GPS_LINK	uint8_t	Qualidade do link GPS	% (0-100)
0x26	FLD_GPS_HDOP	uint16_t	HDOP (precisão horizontal)	×100

## IMU / Atitude

ID (Hex)	Constante	Tipo	Descrição	Unidade

0x31	FLD_PITCH	float	Ângulo de inclinação	radianos
0x32	FLD_YAW	float	Ângulo de guinada	radianos
0x33	FLD_VX	float	Velocidade linear X	m/s
0x34	FLD_VY	float	Velocidade linear Y	m/s
0x35	FLD_VZ	float	Velocidade linear Z	m/s
0x36	FLD_HEADING	float	Rumo magnético	graus
0x37	FLD_ROLL_RATE	float	Velocidade angular roll	°/s
0x38	FLD_PITCH_RATE	float	Velocidade angular pitch	°/s
0x39	FLD_YAW_RATE	float	Velocidade angular yaw	°/s

## Estado de Voo

0x42	FLD_VEL_GPS	float	Velocidade GPS	m/s
0x43	FLD_VEL_CALC	float	Velocidade calculada por fusão	m/s

## Temperatura

ID (Hex)	Constante	Tipo	Descrição	Unidade

## Sistema

ID (Hex)	Constante	Tipo	Descrição	Unidade

## Failsafe

ID (Hex)	Constante	Tipo	Descrição	Unidade
0xA1	FLD_FS_REASON	uint8_t	Razão do failsafe	FailsafeReason enum
0xA2	FLD_FS_ACTION	uint8_t	Ação em curso	FailsafeAction enum
0xA3	FLD_FS_STATE	uint8_t	Estado do failsafe	1=ativo, 0=inativo

## Vídeo

ID (Hex)	Constante	Tipo	Descrição	Unidade
0xB0	FLD_VIDEO_FRAME_ID	uint16_t	Número do frame	-
0xB1	FLD_VIDEO_CHUNK_ID	uint8_t	Índice do chunk	-
0xB2	FLD_VIDEO_TOTAL	uint8_t	Total de chunks neste frame	-
0xB3	FLD_VIDEO_PAYLOAD	uint8_t[]	Payload de vídeo	bytes

# 3. ENUMS AUXILIARES

## SystemState

Valor	Constante	Descrição
0	SYS_STATE_BOOTING	Arranque do sistema
1	SYS_STATE_INIT	Inicialização
2	SYS_STATE_IDLE	Pronto, à espera
3	SYS_STATE_ARMED	Armado
4	SYS_STATE_FLYING	Em voo
5	SYS_STATE_FAILSAFE	Modo segurança
6	SYS_STATE_ERROR	Erro crítico
7	SYS_STATE_SHUTDOWN	Desligamento

## FlightMode

Valor	Constante	Descrição
0	MODE_MANUAL	Controlo direto
1	MODE_STABILIZE	Estabilização angular
2	MODE_ALT_HOLD	Manutenção de altitude
3	MODE_POSHOLD	Manutenção de posição
4	MODE_AUTO	Navegação autónoma
5	MODE_RTL	Return To Launch

## FailsafeReason

Valor	Constante	Descrição
0	FS_REASON_NONE	Sem failsafe
1	FS_REASON_LOST_LINK	Perda de comunicação
2	FS_REASON_LOW_BATTERY	Bateria baixa
3	FS_REASON_GPS_LOST	Perda de GPS
4	FS_REASON_IMU_ERROR	Erro IMU
5	FS_REASON_MANUAL_TRIGGER	Ativação manual

## FailsafeAction

Valor	Constante	Descrição
0	FS_ACTION_NONE	Nenhuma ação
1	FS_ACTION_HOVER	Pairar
2	FS_ACTION_RTL	Regressar à origem
3	FS_ACTION_LAND	Aterrar
4	FS_ACTION_CUT_THROTTLE	Cortar motores
5	FS_ACTION_DISARM	Desarmar

## PriorityLevel

Valor	Constante	Descrição
0	PRIORITY_SUPER_CRITICAL	Failsafe, emergência
1	PRIORITY_CRITICAL	Comandos de voo
2	PRIORITY_HIGH	Telemetria crítica
3	PRIORITY_MEDIUM	Telemetria normal
4	PRIORITY_LOW	Debug, shell
5	PRIORITY_SUPER_LOW	Vídeo, dados extensivos

# 4. EXEMPLOS DE MENSAGENS

## Exemplo: MSG_TELEMETRY (telemetria normal)

```cpp
TLVMessage msg;
msg.msgID = MSG_TELEMETRY;
msg.tlvCount = 6;

addTLVFloat(&msg, FLD_ROLL, 2.34f);
addTLVFloat(&msg, FLD_PITCH, -1.23f);
addTLVFloat(&msg, FLD_HEADING, 145.67f);
addTLVFloat(&msg, FLD_ALT_BARO, 12.34f);
addTLVFloat(&msg, FLD_BATT_V, 12.34f);
addTLVUint8(&msg, FLD_MODE, MODE_STABILIZE);
´´´

## Exemplo: MSG_COMMAND (comando da GS)

```cpp
TLVMessage msg;
msg.msgID = MSG_COMMAND;
msg.tlvCount = 2;

addTLVUint8(&msg, CMD_SET_MODE, MODE_ALT_HOLD);
addTLVFloat(&msg, CMD_SET_ALT_TARGET, 50.0f);
´´´

## Exemplo: MSG_SHELL_CMD (comando shell)

```cpp
TLVMessage msg;
msg.msgID = MSG_SHELL_CMD;
msg.tlvCount = 1;

const char* cmd = "get_roll";
addTLV(&msg, 0x01, cmd, strlen(cmd));
´´´
